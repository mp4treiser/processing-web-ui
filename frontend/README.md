# Frontend - Deal Processing UI

React + TypeScript приложение для системы управления финансовыми сделками.

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env`:
```
VITE_API_URL=http://localhost:8000
```

3. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173
